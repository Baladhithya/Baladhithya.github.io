# Terminal-themed Portfolio Website

This is a modern, responsive portfolio website with a retro terminal theme. The website is built using React and TypeScript, featuring:

- Terminal-inspired black and white design with green accents
- Interactive command-line style sections
- Responsive layout for all device sizes
- Sections for About, Education, Skills, Experience, Projects, Awards, and Contact

## Key Technologies

- React with TypeScript
- Tailwind CSS for styling
- Vite for fast builds and development
- Express backend

## Project Structure

- `client/`: Frontend React application
  - `src/components/`: UI components for each section
  - `src/data/`: Data files containing portfolio information
  - `src/assets/`: Icons and images
  - `src/lib/`: Utility functions
  - `src/hooks/`: Custom React hooks

- `server/`: Simple Express server
  - `index.ts`: Entry point for the server
  - `routes.ts`: API routes
  - `storage.ts`: Data storage functionality
  - `vite.ts`: Vite configuration for the server

- Configuration files:
  - `vite.config.ts`: Vite configuration
  - `tailwind.config.ts`: Tailwind CSS configuration
  - `theme.json`: Theme configuration
  - `tsconfig.json`: TypeScript configuration
  - `postcss.config.js`: PostCSS configuration
